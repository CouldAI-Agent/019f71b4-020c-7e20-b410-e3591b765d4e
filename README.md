# Karodiya Jeweller

A Flutter application for Karodiya Jeweller shop, featuring gold and silver rates, jewelry items, and a sales billing system.

## Features
- **Welcome Screen:** Displays live/mock gold and silver rates, jewelry item images, and shop contact information (Khilchipur, Contact: 9109000391).
- **Billing System:** Generate sales bills by entering customer name, mobile number, location, and adding items to calculate the total amount.
- **Cross-Platform:** Works seamlessly on mobile, tablet, and web browsers.

## Getting Started

### Prerequisites
- Flutter SDK (latest stable version)
- Dart SDK

### Running the App
1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Run the app using:
   ```bash
   flutter run
   ```
